## 🏷️ Asper Brand Context

All prompts in this library are compatible with the **"Ivory & Gold"** design system:

```css
--soft-ivory:   #F8F8FF  /* Global canvas */
--maroon:       #800020  /* Primary actions */
--shiny-gold:   #C5A028  /* The "Gold Stitch" — hover states */
--dark-charcoal: #333333 /* Body text */
```

**Typography:** Playfair Display (headings) · Montserrat (body) · Tajawal (Arabic/RTL)

## 📝 Prompt Format

Every prompt file uses this structure:

```markdown
---
name: [Name]
description: [One-line description]
model: sonnet
---

# [Title]

**Category:** [category]
**Difficulty:** Beginner/Intermediate/Advanced
**Tags:** #tag1 #tag2

## Description
## Prompt
## Example Usage
## Sample Results
## Notes
```
